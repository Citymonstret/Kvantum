Introduction

/* First Piece of Description */ IntellectualServer is our own handcrafted webserver, written entirely in Java. It is not meant to replace your traditional webserver (such as Nginx), but it is fully capable of being used on it's own - If you so desier. We initially created this piece of software, as we needed a simple way to use java to serve dynamic content on the web. We found this usable in more situations than the one we first imagined. That lead us to continue developing the software, into what it is today.

/* Some bragging */ IntellectualServer is made to support both static and dynamic content, with built in caching and back-end speed optimization. We are proud to present what we think is the new generation of Java-based web development.

/* System Requirements */

System Requirements

Minimum

  • OS: Any OS that supports the Java virtual machine
  • RAM: 128MB RAM allocated to the runtime
  • Java Version: Java 1.7

Recommended

  • OS: Debian based OS, Windows 7+, CentOS
  • RAM: 512MB RAM allocated to the runtime
  • Java Version: Java 1.8
/* Showcase of the crush engine */

Crush Engine

This site is served using IntellectualServer. IntellectualServer has a custom templating system built in, along with some custom syntax that is used to enhance the HTML experience.


Crush comes with these syntax features:

  • Comments /* Comment */
  • Variables with filters {{provider.variable || filter}}
  • Foreach loops {#foreach provider.variable -> temp}
  • If statements (and reverse if statements) {#if variable}{/if}
  • File inclusion {{include:./file/name}}
  • Meta variables
