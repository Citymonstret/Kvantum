IntellectualServer & CrushTemplate

CrushTemplate is our very own unique template engine; it's shipped natively with IntellectualServer!

So... what can it do?

Well, it has the ability to:

  • Include files
  • Access variables & and config files
  • Use filters on those variables
  • Use if statements with the variables
