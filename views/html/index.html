IntellectualServer & CrushTemplate

CrushTemplate is our very own unique template engine; it's shipped natively with IntellectualServer!

So... what can it do?

Well, it has the ability to:

  • Include files
  • Access variables & and config files
  • Use filters on those variables
  • Use if statements with the variables
Hello {{session.username || UPPERCASE}},
It's currently {{system.time || LOWERCASE}} and the document is named {{document.name}}!
The server is running on port {{cfg.server@port}} and the hostname is {{cfg.server@hostname}}
At the time of reading this we support these filters: {{system.filters || LIST}} and the one used was LIST!
